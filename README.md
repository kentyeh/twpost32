# twpost32
台灣地址郵遞區號查詢

執行 mvn wro4j:run 先產出 css 與 javascript 的最小化檔案後

執行 mvn jetty:run & 後開啟 [http://localhost:8080/twpost32/](http://localhost:8080/twpost32/) 進行地址查詢郵遞區號
